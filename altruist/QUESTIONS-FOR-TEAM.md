# Вопросы команде Altruist для гайда v1.3

Гайд: [altruist-ai-assistant-guide.md](altruist-ai-assistant-guide.md).
Ниже — информация, которой нет в публичных источниках (sensors.social, вики,
README прошивки). Без неё в гайде стоят заглушки «обратитесь в поддержку».
Ответы позволят ИИ-ассистентам закрывать эти вопросы самостоятельно, разгружая
поддержку.

## 1. Индикация статуса — ✅ частично закрыто (гайд v1.4)

Таблица LED для Urban ESP32-C6 взята из README прошивки
(зелёный/синий/красный, вспышки после отправки). Остались вопросы:

- Insight: точная цветовая шкала LED-периметра (какой цвет = какой уровень
  качества воздуха, по какому показателю);
- legacy Urban на ESP32-C3: есть ли индикация и какая.

## 2. Физическая процедура сброса — ✅ закрыто (гайд v1.4)

Из README прошивки: Urban C6 — кнопка reset >10 с на работающем = сброс Wi-Fi
(identity сохраняется), кнопка при включении = полный factory reset; Insight —
`SET`+`DOWN` 4 с / при включении. Уточнить осталось только: где физически
расположена кнопка reset на корпусе Urban и подписана ли она.

## 3. Обновление прошивки — ✅ частично закрыто (гайд v1.4)

OTA автоматически на Stable-канале; версия и канал видны на статусной
странице. Остались вопросы:

- Публичный URL веб-флешера (webflasher) для ручной прошивки?
- Процедура входа в режим прошивки по USB (аналог «зажать BOOT → подключить»)?

## 4. Кнопки и экраны Insight — ✅ частично закрыто (гайд v1.4)

Кнопки описаны (UP/DOWN экраны, графики, SET — сон, комбо сбросов). Остались
вопросы:

- какие экраны есть на e-ink и что на каждом показывается;
- регулировка LED-периметра: яркость/ночной режим (в веб-конфиге есть только
  полное отключение индикации?).

## 5. Обслуживание и ресурс датчиков

- Ресурс SDS011 (часы наработки) и признаки деградации;
- нужно ли чистить датчик пыли / трубку забора воздуха, как часто и как;
- калибровка SCD41 (CO2): есть ли автокалибровка (ABC), нужно ли
  проветривание для базовой калибровки;
- можно ли заменить датчик самостоятельно, продаются ли запчасти.

## 6. Комплектации и характеристики

- Входит ли RadSens (радиация) в какие-то серийные комплектации, или это
  только DIY-опция прошивки;
- габариты и вес Urban / Insight / UV Cover;
- гарантийный срок и процедура возврата;
- утилизация (WEEE) для продаж в ЕС.

## 7. Локальный API

Home Assistant опрашивает устройство по HTTP локально. Есть ли документация
эндпоинтов (например, JSON с текущими показаниями), которую можно дать
продвинутым пользователям без Home Assistant?

## 8. Приватность координат

Подтвердите рекомендацию из гайда: смещение точки на ~100 м допустимо и не
ломает ничего на карте? Есть ли планы встроенного размытия координат (как у
PurpleAir)?

## 9. Короткий стабильный URL под доменом проекта

Сейчас QR ведёт на raw.githubusercontent.com — он длинный (QR плотнее) и
блокируется в некоторых регионах. Можно ли завести редирект вида
`sensors.social/ai` (302 на актуальную версию гайда)? Плюсы: QR проще и
надёжнее, файл можно переносить/версионировать без перепечатки кодов на
упаковке, работает в регионах, где GitHub raw недоступен.
